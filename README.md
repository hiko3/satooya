# **Satooya**

## アプリ概要 
ペットの里親を募集できるアプリです。

## 作成した経緯
犬猫の殺処分は年間10万匹とも言われています。
ペットが高齢だから、吠えるから、引っ越すから、と様々な理由で行政に持ち込まれ
殺処分されている現実があります。
そんな恵まれない命を救いたいと思い、「satooya」を作成しました。

## リンク
https://satooya.work/

## 機能一覧
- ユーザー関係
  - 新規登録、ログイン、ログアウト
  - プロフィール、登録情報編集

- 投稿関係
  - 新規投稿、編集
  - 一覧表示、詳細表示、編集、削除

- お気に入り機能
  - 投稿をお気に入りに追加、お気に入りから削除（ajax）
  - お気に入りした投稿表示

- 検索・ソート
  - カテゴリ検索
  - キーワード検索
  - 作成日 or 募集期限日でソート

- お問い合わせ
  - 募集しているユーザーへのお問い合わせ
